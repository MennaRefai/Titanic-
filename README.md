# Titanic-
Built a binary classification model to predict Titanic passenger survival. Performed EDA and data preprocessing, then trained multiple algorithms including Logistic Regression (79.9% accuracy), SVM (81.6% accuracy), and Decision Tree (77.7% accuracy). Evaluated models using accuracy, confusion matrix, and classification report.
